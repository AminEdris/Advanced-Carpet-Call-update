Advanced Carpet Call - Help Guide

Introduction

Advanced Carpet Call is designed to help Carpet manufacturer calculate carpet costs, materials, and other key details quickly and accurately.

Main Features

  • Estimation of every carpet element Pile,Warp,Weft,Ground
  • Flexible adjust of calculation of most weave type and structure
  • Calculate pile weight per frame according to design %,required loom running width and min. cone weight
  • Save calculation and print out
  • Possibility to fine correct the calculations of every element in case of a difference from the actual
  • Carpet cost estimation
  • Calculate margin and final price
  • Material and yarn calculations
  • Unit conversions
  • All calculations are automated in case of any change at single inupt to show instant result
  • Helpful for R&D specs estimations
  • Update checking and version control

How to Use

Launch the software and enter the required parameters such as carpet type, materials, or density. Press the appropriate calculation button.

Settings & Updates

You can check for updates through the built-in update feature if enabled.

Support & Contact

For help or feedback, please contact the developer or refer to the project repository.

Start calculations

Input main carpet data

  • input Code, Carpet Name, reed, picks/cm, type of structure
  • choose Others at weave struc. to set the number of weft carry pile in case it is special weave
  • input latex added in grams and price per kilo
  • Input total number of frames this number is important as it will be used to check the right frame inputs at weave A,B,C and D

choose weave structure

  • Choose on of the main weave structure
  • choose Others at weave struc. to set the number of weft carry pile in case it is special weave

Manual Weave Structure Adjust

  • Choose Number of weft carry pile
  • Choose Number of weft that are not carry pile
  • Total weft will be Calculated automatically
  • The percent of carry pile will be calculated accordingly
